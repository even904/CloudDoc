# 7 时序差分方法

## 7.1 Motivating Example

- 三个问题：
    - mean estimation
        - 求解
        $$w=\mathbb{E}[X]$$
        基于随机变量X的部分iid样值$\{x\}$
        - 使用RM算法求解
        - 首先写为$g(w)=w-\mathbb{E}[X]$，将其转变为求根问题
        $$g(w)=0$$
        - 带噪声观测值为
        $$\tilde{g}(w,\eta)=w-x=(w-\mathbb{E}[X])+(\mathbb{E}[X]-x)\doteq g(w)+\eta$$
        - 根据RM算法求解，有
        $$w_{k+1}=w_k-\alpha_k\tilde{g}(w_k,\eta_k)=w_k-\alpha_k(w_k-x_k)$$
    - 复合型:估计函数$v(X)$的均值
        $$w=\mathbb{E}[v(X)]$$
        基于随机变量X的部分iid样值$\{x\}$
        - 首先写为
        $$g(w)=w-\mathbb{E}[v(X)] \\
        \tilde{g}(w,\eta)=w-v(x)=(w-\mathbb{E}[v(X)])+(\mathbb{E[v(X)]}-v(x))\doteq g(w)+\eta$$
        - 转变为求根问题
        $$w_{k+1}=w_k-\alpha_k \tilde{g}(w_k,\eta_k)=w_k-\alpha_k[w_k-v(x_k)]$$
    - 更复杂一点
        $$w=\mathbb{E}[R+\gamma v(X)]$$
        其中$R,X$是随机变量，$\gamma$是常数，$v(\cdot)$是函数
        - 假设我们可以获取随机变量$X$和$R$的样值$\{x\}$和$\{r\}$，定义
        $$\begin{aligned}  g(w)&=w-\mathbb{E}[R+\gamma v(X)] \\
            \tilde{g}(w,\eta)&=w-[r+\gamma v(x)] \\
            &=(w-\mathbb{E}[R+\gamma v(X)])+(\mathbb{E}[R+\gamma v(X)]-[r+\gamma v(x)]) \\
            &\doteq g(w)+\eta
        \end{aligned}
        $$
        - 然后，问题转变为求根问题$g(w)=0$，相应的RM算法是
        $$w_{k+1}=w_k-\alpha_k\tilde{g}(w_k,\eta_k)=w_k-\alpha_k[w_k-(r_k+\gamma v(x_k))]$$

## 7.2 TD learning of state values

> TD Learning 指的是一大类的RL算法
> 本节课介绍一种特定的（原始的）TD Learning，用于估计state value，具有特定的表达式
   
- TD算法特性
    - 用于做policy evaluation，给定策略，计算state value
    - 不能估计action value
    - 不能寻找最优policies

- TD算法描述
    - $\textcolor{blue}{(s_0,r_1,s_1,...,s_t,r_{t+1},s_{t+1},...)}$ 或 $\textcolor{blue}{\{s_t,r_{t+1},s_{t+1}\}}$均由给定策略$\pi$生成
    - TD learning算法表达式如下：
    $$\begin{aligned}
        \textcolor{red}{v_{t+1}(s_t)}&\textcolor{red}{=v_t(s_t)-\alpha_t(s_t)[v_t(s_t)-[r_{t+1}+\gamma v_t(s_{t+1})]]} \\
        \textcolor{red}{v_{t+1}(s)}&\textcolor{red}{=v_t(s), \forall s\neq s_t}
    \end{aligned}$$
    其中$t=0,1,2,...$，因此，$v_t(s_t)$是$v_\pi(s_t)$的state value估计值，$\alpha_t(s_t)$是在时刻$t$时，$s_t$的学习率
    - 在$t$时刻，只有被访问过的状态$s_t$的值发生更新，而未访问的状态$s\neq s_t$的值保持不变
    - 第二个式子，一般会隐藏不写，但一般隐含此条件
    - 第一个式子的详细解释：
    $$\underbrace{v_{t+1}(s_t)}_{\text{new estimate}}=\underbrace{v_t(s_t)}_{\text{current estimate}}-\alpha_t(s_t)\left[\overbrace{v_t(s_t)-[\underbrace{r_{t+1}+\gamma v_t(s_{t+1})}_{\text{TD target } \bar{v}_t}]}^{\text{TD error }\delta_t}\right]$$
    这里
    $$\bar{v}_t\doteq r_{t+1} + \gamma v(s_{t+1})$$
    称为TD target
    $$\delta_t\doteq v(s_t)-[r_{t+1}+\gamma v(s_{t+1})]=v(s_t)-\bar{v}_t$$
    称为TD error
    显然新的估计$v_{t+1}(s_t)$是当前估计$v_t(s_t)$和TD error的组合
    - $\bar{v}_t$为什么称为TD target？
        - 因为该算法将$v(s_t)$朝$\bar{v}_t$的方向改进
        $$
        \begin{aligned}  
        &v_{t+1}(s_t)=v_t(s_t)-\alpha_t(s_t)-\alpha_t(s_t)[v_t(s_t)-\bar{v}_t] \\
        \xRightarrow{} \quad &v_{t+1}(s_t)-\textcolor{blue}{\bar{v}_t} = v_t(s_t)-\textcolor{blue}{\bar{v}_t}-\alpha_t(s_t)[v_t(s_t)-\bar{v}_t] \\
        \xRightarrow{} \quad &v_{t+1}(s_t)-\textcolor{blue}{\bar{v}_t} = [1-\alpha_t(s_t)][v_t(s_t)-\textcolor{blue}{\bar{v}_t}] \\
        \xRightarrow{} \quad &|v_{t+1}(s_t)-\textcolor{blue}{\bar{v}_t}|=|1-\alpha_t(s_t)||v_t(s_t)-\textcolor{blue}{\bar{v}_t}|
        \end{aligned}
        $$
        由于$\alpha_t(s_t)$是个非常小的整数，我们有
        $$0\lt 1-\alpha_t(s_t) \lt 1$$
        因此
        $$|v_t(s_t)-\textcolor{blue}{\bar{v}_t}|\leq|v_t(s_t)-\textcolor{blue}{\bar{v}_t}|$$
        意味着$v(s_t)$正朝$\bar{v}_t$前进
    - TD error的解释？
        $$\delta_t=v(s_t)-[r_{t+1}+\gamma v(s_{t+1})]$$
        - 两个连续时间步之间的差值
        - 反映了$v_t$到$v_\pi$之间的欠缺，可以记
        $$\delta_{\pi,t}\doteq v_\pi(s_t)-[r_{t+1+\gamma v_\pi(s_{t+1})}]$$
        注意到
        $$\mathbb{E}[\delta_{\pi,t}|S_t=s_t]=v_\pi(s_t)-\mathbb{E}[R_{t+1}+\gamma v_\pi(S_{t+1})|S_t=s_t]=0$$
        如果$v_t=v_\pi$，那么$\delta_t$应为0
        然而，如果$\delta_t$非零，那么$v_t$不等于$v_\pi$
        - TD error可解释为新信息，从经验$(s_t,r_{t+1},s_{t+1})$获得

## 7.3 TD 算法收敛性，与MC算法比较

- TD算法解决的问题：
    - 在没有模型的情况下解决贝尔曼公式
    - 引入贝尔曼公式的新表达式
    $\pi$的state value的定义是
    $$v_\pi(s)=\mathbb{E}[R+\gamma G|S=s],\quad s\in \mathcal{S} \tag{4}$$
    其中$G$是discounted return。由于
    $$\mathbb{E}[G|S=s]=\sum_a\pi(a|s)\sum_{s'}p(s'|s,a)v_\pi(s')=\mathbb{E}[v_\pi(S'|S=s)]$$
    其中$S'$是下一个状态，我们可以将(4)式重写为
    $$\textcolor{blue}{v_\pi(s)=\mathbb{E}[R+\gamma v_\pi(S')|S=s],\quad s=\mathcal{S}} \tag{5}$$
    方程(5)是贝尔曼方程的另一种表达式，有时称为**贝尔曼期望方程**
    - 使用RM算法求解式(5)的贝尔曼方程
    定义
    $$g(v(s))=v(s)-\mathbb{E}[R+\gamma v_\pi(S')|s]$$
    可以将(5)重写为
    $$g(v(s))=0$$
    因为我们只能得到$R$和$S$的样值$r$和$s'$，带噪声观测值是：
    $$\begin{aligned}
        \tilde{g}(v(s))&=v(s)-[r+\gamma v_\pi(s')] \\
        &=\underbrace{\left(v(s)-\mathbb{E}[R+\gamma v_\pi(S')|s]\right)}_{g(v(s))}+\underbrace{\left(\mathbb{E}[R+\gamma v_\pi(S')|s]-[r+\gamma v_\pi(s')]\right)}_\eta
    \end{aligned}$$
    因此，解方程$g(v(s))=0$的RM算法就是
    $$\begin{aligned}
    v_{k+1}(s)&=v_k(s)-\alpha_k\tilde{g}(v_k(s))\\&=v_k(s)-\alpha_k\left(v_k(s)-\left[r_k+\gamma v_{\textcolor{blue}{\pi}}(s_k^{\prime})\right]\right),\quad k=1,2,3,\ldots
    \end{aligned} \tag{6}$$
    其中 $v_k(s)$ 是 $v_\pi(s)$在第$k^{th}$步的估计值，$r_k$，$s_k'$是$R$,$S'$在第$k^{th}$步的采样值
    (6)中的RM算法有两个值得注意的假设
        - 需要有第 $k$步的$\{(s,r,s')\}$ $k=1,2,3,...$
          在一个episode中，$\{(s,r,s')\}$替换为$\{(s,r_{t+1},s_{t+1})\}$，如果恰巧访问到$s$就更新，否则不更新
        - 我们假设$v_\pi(s')$对所有$s'$均已知
          $v_\pi(s')$使用一个估计值来替代，因为我们不能提前知道它
    - TD算法收敛性定理
    当$t\to \infty$时，若$\sum_t\alpha_t(s)=\infty$且$\sum_t\alpha_t^2\lt\infty$，则对于所有的$s\in\mathcal{S}$，$v_t(s)$ 以概率1依概率收敛到$v_\pi(s)$。
        - 该定理证明了TD算法对于一个给定策略$\pi$能够找到state value
        - $\sum_t\alpha_t(s)=\infty$且$\sum_t\alpha_t^2\lt\infty$必须对仍以$s\in \mathcal{S}$均成立，即任意一个状态必须被访问足够多次
        - 工程上，学习率$\alpha$一般选择一个很小的常数，而非让其收敛到0，因为需要保留有效经验

|TD/Sarsa learning|MC learning|
|:---:|:---:|
|**Online**|**Offline**|
|**Continuing tasks**|**Episodic tasks**|
|**Bootstrapping** Require initial guesses|**Non-bootstrapping** No need initial guess|
|**Low estimation variance** Fewer random variables, for instance, Sarsa requires $R_{t+1},S_{t+1},A_{t+1}$|**High estimation variance** To estimate $q_\pi(s_t,a_t)$，we need samples of $R_{t+1}+\gamma R_{t+2}+\gamma^{2}R_{t+3}+\ldots$ Each episode length is L, there are ${\|\mathcal{A}\|}^L$ possible episodes|
